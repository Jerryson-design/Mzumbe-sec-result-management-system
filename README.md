## 📌 Project Description
The **School Management Information System (MIS)** is a web-based system built using HTML, CSS, and JavaScript.  
It is designed to manage basic school operations such as student records, attendance, and results.

This system runs **offline** using browser local storage (no database required).

---

## 🚀 Features

- 🔐 User Login & Registration (Admin/Teacher)
- 👨‍🏫 Dashboard for management
- 📝 Student registration & records management
- 📊 Results entry and viewing
- 📅 Attendance tracking system
- 🖨️ Print student results
- 📩 Send notifications (simulation to Academic Master)
- ❌ Logout & cancel options

## ⚙️ How to Run

1. Download or clone this repository
2. Place all files in one folder
3. Open `index.html` in any browser (Chrome recommended)
4. Start using the system offline

---

## 💾 Data Storage
- Uses browser **localStorage**
- No backend or database required
